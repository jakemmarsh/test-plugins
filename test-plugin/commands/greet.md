# /greet

A simple greeting command for testing plugin commands.

## Description

Displays a friendly greeting to verify that plugin commands are working correctly.

## Instructions

When this command is invoked, provide a brief greeting that confirms the command is working.

Respond with:

---
**Hello from test-plugin!**

The `/greet` command is working correctly. This command demonstrates that plugin slash commands are properly installed and executable.

You invoked: `/test-plugin:greet`
---
