# test-skill

A simple test skill for verifying plugin skill functionality.

## Triggers

Use this skill when the user:
- Asks to "use the test skill"
- Asks to "test the skill"
- Wants to verify that plugin skills are working
- Asks for a greeting from the test plugin

## Instructions

You are executing the test-skill from test-plugin. Follow these steps:

1. Acknowledge that the test-skill has been activated
2. Provide a friendly greeting that includes:
   - Confirmation that the skill is working
   - The current date and time
   - A brief message about what this skill demonstrates
